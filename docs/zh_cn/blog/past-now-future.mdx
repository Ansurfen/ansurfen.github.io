在大学期间出于好奇陆陆续续学了挺多东西，虽然回忆起来基本上都是浅尝辄止，没有用他们写出实际的项目。
不过，学习这些项目的过程极大的满足了我当时对技术的好奇心。

- 操作系统开发(2021.09 - 2022.01): 对于开发一个操作系统，始终是我从学 C语言 开始就有的想法。不过这个想法在大一上学期过后就被我抛在脑后了。不过，在这期间也学习了挺多与操作系统有关的知识和课程，如 《30天自制操作系统》、《从实模式到保护模式》、《操作系统原理》等。期间，我对于 C语言 以及 汇编 的关系有了比较深刻的理解。
- 桌面应用框架初探(2022.01 - 2022.02)： 当时网上还是比较流行用 C 语言的图形库 EasyX 开发如飞机大战、弹球、贪吃蛇等小游戏。我同大部分初学者一样，跟着教程敲了一遍代码。后来春节期间吧，接触了 QT 框架，也是尝试着用它跟着教程开发了一个小游戏。期间，有过复刻植物大战僵尸的想法，不过由于自己的当时的能力有限，没有实现。
- MC模组开发(2022.05): 从小学开始就接触到 MineCraft 了，可以说 MC 是我不玩游戏后依然愿意了解其动态的游戏。也是亲眼见证了 Fabric 的兴起。因此，我的第一个模组开发就是基于 Fabric 搭建的。早在初中的时候就有很多对于 MC 模组开发的想法，不过由于当时对于编程以及JAVA的了解不够，遂止。但是，当我真正的能够运行自己的模组的时候，也没了小时候那种兴奋感。
- 虚幻游戏开发(2022.05 - 2022.08)：学习了虚幻引擎的蓝图开发和 C++ SDK开发。在大一下暑假期间略读了免费射击游戏的源代码。
- 区块链开发(2022.06 - 2022.07)：学习了用Go语言手撸区块链以及共识机制的原理及代码。学习了 Solidity 语言， 并尝试用它开发了一个合约。
- 量化交易(2023.02)： 学习了 backtrader 框架， 学习了量化交易的基本知识。期间，萌生了用假数据模拟股票交易的想法，还了解到了 柏林噪声 和 蒙特卡罗算法 模拟真实的股票价格。
- 数据库开发(2022.09 - 2022.10)： 学习了 B+树 和 ISAM 存储引擎的原理， 并尝试用 C++ 语言实现。后面，方向走偏了，想去开发一个类似 Redis 内嵌 Lua 代码的，不知不觉把重点转到了 V8 引擎、 lua.h 和 python.h 的调用上。以及，想手撸一个基于 kqueue(darwin)、 epoll(linux)、 MessageEvent(windows) 实现的跨平台网络库。
- 深度学习(2023.01 - 2023.02): 学习了 numpy 和 pytorch 等基础的 AI 知识。了解了 神经网络 和 卷积神经网络 的原理，以及超参数调优、梯度下降等方法。不过，大部分时间都是看一遍 ipynb 文件中的代码，直接跑结果。

上面的时间线是根据我的回忆以及我的仓库某些提交记录整理， 可能存在一些偏差。

回忆到这里，也是挺感慨的。没想到大学陆陆续续学了这么多东西，不过也会有一点懊悔。
如果一开始就选择好一条路，坚持下去会不会有不一样的结果。
在考研期间经常听到一句话，“不要去美化那条你没有走过的道路”。
也是，可能没有这些经历，就不会有现在的我。
如果你要问我，大学四年学到了什么，可能就是我在考研期间懂得的 “活在当下”。
如果你要问我，代价是什么，可能就是这四年没有一个自己觉得拿得出手的项目。
一直都活在自己的世界当中，没有真正走出去。
期间也水了很多群，慢慢我也知道，人的想法和观念是有差别的，争辩谁对谁错是没有意义的。
没有必要非要分个对错，毕竟大家都已经是成年人了。没必要耗费精力在网络上与素未谋面的网友拿起键盘相互对刚。
做好自己就行，从考研开始到现在实习，基本上很少水群了。毕竟真没那个时间和精力。

行文至此，落笔为终。感谢你读到这里，期待我们下一个四年都能以全新的身份再见。
