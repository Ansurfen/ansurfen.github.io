利用yock代替bat和bash脚本

2023-07-17
yock
932 词

  • yock

注意: 由于GNU命令过于庞杂，因此对于相关函数以代码和注释的形式穿插介绍。

原生执行

调用终端直接执行，对于不同平台相同命令不同flag并不具备适配的能力。因此，如果你想获得更好的跨平台能力，最好仅使用原本就跨平台的命令。例如，支持跨平台的命令: go mod, npm; 不支持跨平台的命令: 操作系统原生命令（tasklist, ps）。

字符串处理

为了实现字符串三剑客跨平台，yock内置第三方实现以代替它们。
原命令 替代
grep => ripgrep
awk => goawk
sed => sd
